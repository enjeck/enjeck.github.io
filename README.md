## Personal Website

This is a personal website built with Next.js and Tailwind CSS.
It has some cool animations and a blog. 
See it live at [enjeck.com](https://enjeck.com).

![enjeck.com](public/images/enjeck.png)
![enjeck.com](public/images/enjeckblog.png)


## Running the project

To run this repo locally:
```
npm install
npm run build
npm run start
```
